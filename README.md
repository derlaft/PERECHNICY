# PERECHNICY

ASM-Programmed ~~massive~~ online multiplayer sandbox

# Building

1. Get gb:

   go get github.com/constabulary/gb/...

2. Make project:

   gb build

3. Run everything:

   ./bin/client ... # client
   ./bin/server # server
   ./bin/cli-runner ... # CLI tool to run PER programs

Run ``. bashrc`` if you want to set your GOPATH correctly
